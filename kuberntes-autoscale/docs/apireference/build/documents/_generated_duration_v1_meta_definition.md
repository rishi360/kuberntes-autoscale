## Duration v1 meta

Group        | Version     | Kind
------------ | ---------- | -----------
meta | v1 | Duration



Duration is a wrapper around time.Duration which supports correct marshaling to YAML and JSON. In particular, it marshals into strings, which can be used as map keys in json.
